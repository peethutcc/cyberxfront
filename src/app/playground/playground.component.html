Redhat

...

Red Hat เป็น Distribution หนึ่งใน Linux ที่ได้รับความนิยมสูงสุดเป็นอันดับหนึ่งในอเมริกา เมื่อเทียบกับ Distribution อื่นๆของ Linux เพราะ RedHat มีโปรแกรมติดตั้งที่ใช้งานง่าย ตลอดจนโปรแกรมเสริมหรือแอพพลิเคชั่นที่น่าสนใจอีกมากมาย

Solaris

...

The Solaris Operating Environment เป็นระบบปฏิบัติการเครือข่ายที่ อยู่ในตระกูลเดียวกับระบบปฏิบัติการ Unix พัฒนาขึ้นโดยบริษัทซัน ไมโครซิสเต็มส์ สามารถรองรับการทำงานแบบเครือข่ายได้เช่นเดียวกับระบบอื่น

...

Kali linux ระบบปฏิบัติการนั่นเอง ซึ่งได้รับการพัฒนามาจาก Linux Debian โดยเป็นระบบปฏิบัติการแบบ Single User และออกแบบมาเพื่อ digital forensice และ penetration testing

Windows Server

...

Windows Server คือ ระบบปฏิบัติการที่ออกแบบโดย Microsoft ซึ่ง Windows Server สนับสนุนการจัดการระดับองค์กร การจัดเก็บข้อมูลแอพพลิเคชันและการติดต่อสื่อสาร Windows Server มีความสำคัญต่อเสถียรภาพความปลอดภัยระบบเครือข่ายและการปรับปรุงระบบไฟล์ต่างๆ

Packet Tracer

...

Docker

...

Docker คือ engine ตัวหนึ่งที่มีการทำงานในลักษณะจำลองสภาพแวดล้อมขึ้นมาบนเครื่อง server เพื่อใช้ในการ run service ที่ต้องการ มีการทำงานคล้ายคลึงกับ Virtual Machine แต่ docker จะใช้ container ในการจำลองสภาพแวดล้อมขึ้นมา เพื่อใช้งานสำหรับ 1 service ที่ต้องการใช้งานเท่านั้น โดยไม่ต้องมีส่วนของ OS เข้าไปเกี่ยวข้องเหมือน Virtual Machines อื่นๆ
